Meu projeto de estudo

Eu me chamo Florêncio e este é um site que irei criar para praticar o que eu for aprendendo no curso The Complete Full-Stack Web Development Bootcamp.

Irei atualizar o site com o conteúdo que eu for aprendendo, aula a aula, no decorrer do curso. Cada atualização no GitHub irá cobrir o conteúdo de uma aula. O que você está vendo aqui é somente a parte de introdução ao HTML. Em breve também irei incrementar o site com CSS e JavaScript.

No decorrer do curso esse site deverá mudar bastante. Ele irá refletir o meu domínio sobre a criação, estruturação e criação de conteúdo para internet.

Introdução ao HTML

A princípio, esse site está usando somente a estrutura mais básica do HTML. Marcadores de título, parágrafos, quebras de linha e uma linha horizontal. Além desses marcadores também acrescentei alnguns outros marcadores necessários para a estruturação do site, como o DOCTYPE, a tag html, head e body. Além disso, acrescentei também a tag meta para definir a codificação de caracteres e a tag title para definir o título do site.

Desenvolvido com Visual Studio Code no ambiente Fedora Linux.
